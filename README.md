# BirdJump 🐦

A Flappy Bird-inspired game built in Unity. Guide your bird through an endless series of obstacles by tapping to jump — how far can you go?

## 🛠️ Built With

- **Unity**
- **C#**

## 🎮 How to Play

- Press **Space** to make the bird jump
- Avoid the pipes — if you hit one, it's game over
- Try to beat your high score!

## 🚀 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/FlowerZone/BirdJump.git
   ```
2. Open **Unity Hub** and click **Add Project**
3. Select the cloned `BirdJump` folder
4. Open the project in Unity (recommended version: *2022.3.27f1*)
5. Press the **Play** button in the Unity Editor to run the game
